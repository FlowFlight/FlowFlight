# FlowFlight

### Opis

Projekt ten ma na celu opracowanie zaawansowanego algorytmu sztucznej inteligencji zaprojektowanego, aby pomóc kontrolerom ruchu lotniczego w podejmowaniu wydajnych i skutecznych decyzji dotyczących sekwencjonowania samolotów do lądowania i odlotu. Głównym celem jest radzenie sobie z zakłóceniami, takimi jak opóźnienia, które mogą zakłócić cały harmonogram, optymalizując w ten sposób ogólne operacje na lotnisku.


### Cele projektu

1. Nasza aplikacja pozwoliłaby liniom lotniczym zaoszczędzić nawet dziesiątki milionów złotych rocznie dzięki zmniejszeniu ilości czasu na staniu samolotów w kolejkach.

2. Nie zlikwidowalibyśmy całkowicie opóźnień i kolejek, bo jest to niemożliwe. Można je jednak znacznie skrócić. 


### Plusy projektu
  - Zaoszczędzenie kosztów
  - Bezpieczeństwo. Zmniejszenie szansy na błąd ludzki.
  - Lepsze doświadczenie pasażerów.

### Jak uruchomić prototyp aplikacji

Zainstaluj pip i npm.

**Windows**
- Uruchom start.cmd.
- Skopiuj link localhost do przeglądarki

**Linux**
- Uruchom start.sh.
- Skopiuj link localhost do przeglądarki
